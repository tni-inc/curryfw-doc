FileWriter

クラス概要

FileWriterはファイルにテキストを書き込むための機能を提供します。

Curry対応バージョン

バージョン 1.2.0 以降

クラスファイル位置

curry/utility/file_writer.php

フィールド

公開されるフィールドはありません。

メソッド

メソッド名 アクセス 概要
__construct public コンストラクタです。
openメソッドも同時に実行されるため、インスタンス生成時には既にファイルが開いた状態になります。
open public 書き込みのためにファイルを開きます。
インスタンス生成時には既に開いている状態のため、通常はこのメソッドを実行することはありません。
closeメソッドで明示的に閉じた後、同一インスタンスで再度書き込みを行う場合にこのメソッドを利用します。
close public ファイルを閉じます。
write public ファイルにテキストを書き込みます。
writeLine public ファイルにテキストを書き込み、最後に改行をコードを付加します。
writeLines public 複数行のテキストを一度に書き込みます。
writeCsv public 2次元配列のデータをCSV形式で書き込みます。
