Response::redirect

public void redirect ( string $url, boolean $exit = true )
指定されたURLへリダイレクトを行います。

パラメーター

名前 型 内容
$url string リダイレクト先URL
$exit boolean リダイレクトのヘッダ送信後にexitするか否か。
省略するとexitします。

返り値

返り値はありません。
